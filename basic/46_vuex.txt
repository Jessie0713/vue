
跨組件的溝通 (vuex)

為何要使用 Vuex (相當於 React 的 Redux)？
    組件之間常跨結構的傳遞狀態，就需要全域的狀態管理
---------------------------------------------------------------------------------
如何引用？
    方法 1. 引入 vue 和 vuex 的 CDN (43_組件之間的溝通3_vuex.html)
    方法 2. (要將目標移到 vue-lessons 資料夾)
        指令: npm i -S vuex  (讓 npm 幫你從雲端下載並安裝 Vuex)
        指令: npm run dev
