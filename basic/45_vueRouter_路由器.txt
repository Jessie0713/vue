
路由器 (vue router)
1. 如果建立了不同的組件，用 v-if, v-else 來選擇頁面(組件)
   也可以用選單的方式選擇頁面，這種操作稱之為 SPA (Single Page Application)
2. 如果點擊某個連結，可以連到該網址的頁面，或是直接打上網址就可以執行該頁面

   以上，vue router可以輕鬆做到
---------------------------------------------------------------------------------
如何引用？ (要將目標移到 vue-lessons 資料夾)
    指令: npm i -S vue-router      (讓 npm 幫你從雲端下載並安裝 Vue Router)
    指令: npm run dev
